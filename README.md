📌 Overview

RoadmapX is a productivity and learning platform designed to help students and developers stay consistent and organized. It combines structured roadmaps with powerful tools like focus sessions, streak tracking, and revision management.

✨ Current Features

- 🔐 User Authentication (Login & Signup)
- 🧭 Structured Learning Roadmaps
- ⏱️ Pomodoro Timer (focus sessions)
- 📊 Progress Tracking
- 🔥 Daily Streak System
- 🔁 Revision Tracking System
- 🌐 Live deployed frontend
- 🔗 Backend API integration

🚧 Features in Progress

- 🤖 AI Chat Assistant
- 📝 Notes & Summarizer
- 🏆 Leaderboard

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript / React
- Backend: Node.js, Express
- Deployment:
  - Frontend: Cloudflare Pages
  - Backend: Render

🌐 Live Demo

👉 https://roadmapx-frontend.pages.dev

🎯 Goal

To build an all-in-one platform that helps users learn effectively, stay consistent through streaks, revise efficiently, and track their overall progress.
